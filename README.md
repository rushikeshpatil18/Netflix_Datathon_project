Problem Statement

Analyze the Netflix dataset to understand patterns in content distribution, including Movies vs TV Shows, genres, and countries.
The project focuses on identifying trends in content growth over time and audience targeting through ratings.
Using data cleaning and visualization techniques, the goal is to generate meaningful insights that reflect Netflix’s content strategy.
These insights can help in making data-driven decisions for content creation and platform expansion.
